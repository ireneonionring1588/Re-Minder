About the Project

We're creating a VR experience designed especially for older adults and anyone concerned about memory loss or Alzheimer's. Set in a tranquil, nature-inspired environment, the space offers gentle interactions—like personalized photo frames, memory games, and interactive objects—that support cognitive function and spark emotional connection.

What You'll Find Inside

Calming Scenery

Calming scenes and gentle audio engage memory recall and reduce anxiety.

Memory Touchpoints

Interactive photo frames and personal visuals that spark emotional connection.

Gentle Exercises

Light cognitive prompts encourage engagement without pressure.

Emotional Support

Designed to feel welcoming, nonjudgmental, and emotionally safe.

How It Works

Re-Minder is designed for ease and comfort. Users can interact with the environment using simple hand gestures or intuitive controller inputs. Whether it’s teleporting across the island or gently grabbing keepsakes, everything is optimized for first-time VR users.
